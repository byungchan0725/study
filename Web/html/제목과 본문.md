### 제목과 본문  

#### 시작 방법 

**!** 를 누르고 엔터를 눌러 아래와 같은 기본 코드를 생성한다.  

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

그리고 위 코드에서 html lang="ko"로 설정하여 컴퓨터가 한글을 읽을 수 있도록 해준다.
